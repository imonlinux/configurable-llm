# Component-level requirements pulled in transitively when this integration's
# platforms import Home Assistant components. The base
# pytest-homeassistant-custom-component install does NOT install per-component
# requirements, so these top-level imports fail without the packages present:
#   - conversation  -> `from hassil.recognize import RecognizeResult`  (hassil, home-assistant-intents)
#   - ai_task -> camera -> `from turbojpeg import TurboJPEG`           (PyTurboJPEG)
# Note: PyTurboJPEG's *import* succeeds without the native libturbojpeg system
# library; that library is only needed to actually decode JPEGs, which the test
# suite never does. So no apt/system package is required.
hassil
home-assistant-intents
PyTurboJPEG
